## GitHub configuration

1. Open **Settings → General** and paste the description above.
2. Keep `main` as the default branch.
3. Add the listed topics through **About → Settings** on the repository page.
4. Keep GitHub Actions enabled so the README badge reflects CI status.
5. Add real images under `docs/screenshots/` using the filenames linked from
   the README.
